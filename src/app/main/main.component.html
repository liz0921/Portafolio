CenfoMarket

Mercado de NFT

Se desarrolló una aplicación web en la asignatura Proyecto 2 en la Universidad Cenfotec con las siguientes tecnologías (.NET, Azure como sistema en la nube, SQL Server para datos persistencia). Algunas herramientas que se integraron fueron Sendgrid, Cloudinary, Twilio y PayPal. La aplicación consistía en un mercado donde los usuarios podían crear sus propios NFT, el usuario también podría poner sus NFT a la venta, en una subasta o recibir ofertas de otros usuarios.

StudyBuddy

Aplicación de estudio

Se desarrolló una aplicación web llamada "Study Buddy" como parte del curso Proyecto 3 en la Universidad Cenfotec, utilizando las siguientes tecnologías (MySQL como base de datos, Angular, Spring Boot y JHipster). La aplicación consistía en una plataforma de estudio donde profesores y estudiantes podían compartir conocimientos. Además, podían compartir materiales educativos relevantes, como videos y archivos PDF, crear salas de estudio y canales de comunicación. El objetivo era proporcionar conocimientos sobre diferentes temas a personas interesadas en ampliar su aprendizaje en un área específica
